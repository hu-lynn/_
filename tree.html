Spinning Christmas Tree

Project Overview

A fun 3D-printed spinning Christmas tree with a stepper motor, Neopixel ring, and Arduino for LAUNCH Engineering Club. The students all made one of these each, learning how to code Arduino, about electronics, and how to make their own project. Pretty simple, but very fun!
